# SimpleMultimeter

#### 介绍
科协第一次训练题软件A题源代码和设计参考
可以独立完成以下几种功能
1. 测量电流
2. 测量电阻（量程为10kΩ-100kΩ）
3. 测量温度（为K型热电偶的精度，能够满足室温和工业量程

#### 软件架构
基于Keil和MDK-ARM软件的设计
